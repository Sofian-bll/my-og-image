Features

8 Template Types

Project, Resource, Task, Aspiration, Machine, Service, System Config, and Epitech Project.

Takumi Renderer

Clean, type-driven layouts via @takumi-rs/core. Pure SVG with Satori — no canvas, no headless browser.

Obsidian Vault Sync

Batch-apply OG URLs across an entire vault via Python scripts. Dry-run mode included.

Docker Ready

Multi-stage Dockerfile + compose. Deploy to any VPS in minutes.

Quick Start

Get up and running in under 60 seconds.

Docker 
docker compose up -d
# Dev with hot reload
docker compose -f docker-compose.dev.yml up -d
Local 
git clone https://github.com/Sofian-bll/my-og-image.git
cd my-og-image
pnpm install
# → http://localhost:3000
pnpm dev
